# GO(lang) Playground

[![CircleCI](https://circleci.com/gh/jcuffney/golang-playground/tree/main.svg?style=svg)](https://circleci.com/gh/jcuffney/golang-playground/tree/main)

## Requirements

- nodejs, npm, go

## Getting Started

- `npm i`
- `go mod download`
- `make dev`

## Commands

`$ make <command>`

| command                  | description                  |
|--------------------------|------------------------------|
| `init`                   | download modules             |
| `init:hard`              | `init` with a fresh `go.sum` |
